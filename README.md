# Academic Homepage Static Template

This is a simple static academic homepage template for GitHub Pages.

## File structure

```text
academic_homepage_static_template/
├── index.html
├── style.css
├── assets/
│   ├── images/
│   │   └── profile-placeholder.svg
│   └── files/
│       └── cv.pdf
└── README.md
```

## How to use

1. Rename the repository to `yourusername.github.io`.
2. Upload all files to the repository.
3. Go to GitHub repository `Settings` → `Pages`.
4. Choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Visit `https://yourusername.github.io`.

## What to edit first

- Replace `Tianyang Wang` with your own name if needed.
- Replace `your_email@example.com`.
- Replace `yourusername` in GitHub and LinkedIn links.
- Replace `assets/images/profile-placeholder.svg` with your own profile photo.
- Replace `assets/files/cv.pdf` with your real CV.
